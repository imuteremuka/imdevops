**_ Repositories _**

--- Create repositories for RPM (redhat package manager) - you can create local repositories or use the Redhat Repositories

**_ yum _**

--- yum search <options> if you are searching for software

--- yum install <options> to install software

--- yum remove <options> removes software packages

--- yum update <options> - compares packages on your system and newer version

--- yum provides <options> more powerful than yum search

--- yum info <info> provides information about packages

--- yum list alll - list all installed packages

**_ yum module _**

--- yum module provides

--- yum module list

--- yum module info

--- yum module install

--- yum module install

**_ yum groups _**

--- **_ yum groups list _** - shows all yum groups available

--- **_ yum groups info <groupname> _** - gives information of a group

--- **_ yum group install <groupname> _** - installs packages of the group

**_ yum history _** show all commands relating to yum and you can also undo

**_ yum update _** update all yum updates
