# Basic Commands

The cli for Linux is one of the most powerful tools. The structure of the command is <command><options>

- pwd
- ls -l
- who am i
- free -m
- df -h
- cat
- find mnt

# Bash Shell

- it is the command interpreter for linux

## Tab Key

    - the tab key is used to auto complete commands

## Command History

- it is used to show commands you have run

<b> The Eclamation Mark </b>

- ! <command history number> to run history number

## Pipping |

- it is used to take results from one command as input to the following command

## Redirection > or >>

- it used to redirects out to a terminal of file
- a single redirects >
- a double redirects >> appends
